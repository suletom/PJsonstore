For any documentation/updated info please visit: https://github.com/suletom/PJsonstore/

This project is minimalistic header library for arduino. The library uses a modified version of jsmn to parse JSON data and it can hold and modify the data in memory. Of course it can write data back to JSON.

The main purpuse was to create a global configuration object for an ESP8266 device to create webUI and store configured data to flash.
